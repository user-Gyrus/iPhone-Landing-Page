# 🍎 iPhone 15 Pro Landing Page

An interactive and visually engaging landing page inspired by Apple's iPhone 15 Pro product showcase.

This project focuses on delivering a premium user experience through modern animations, smooth transitions, responsive layouts, and immersive product presentation.

## ✨ Features

* 🎨 Modern Apple-inspired UI
* 📱 Fully Responsive Design
* 🎥 Embedded Product Videos
* ⚡ Smooth Scroll Animations
* 🖥️ Interactive Product Showcase
* 🌍 Cross-Device Compatibility

## 🛠️ Tech Stack

| Category     | Technologies |
| ------------ | ------------ |
| Frontend     | React.js     |
| Animations   | GSAP         |
| 3D Rendering | Three.js     |
| Build Tool   | Vite         |
| Deployment   | Vercel       |
